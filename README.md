# Portfolio

Personal portfolio built as a full-stack application with server-side rendering, showing projects, skills and a form to contact me.

## Tech Stack

| Layer     | Technology                        |
|-----------|-----------------------------------|
| Frontend  | Angular 20, TypeScript, Angular CDK |
| Backend   | Node.js 20, Express 5, TypeScript |
| Rendering | Angular Universal (SSR)           |

## Prerequisites
- Node.js >= 20
- npm >= 10
- Angular CLI >= 20

## Installation

```bash
git clone https://github.com/itisalicia/portfolio.git
```

```bash
cd portfolio
```

# Frontend

```bash
cd frontend
npm install
```
# Backend

```bash
cd backend
npm install
```
## Deployment

Deployed with vercel
